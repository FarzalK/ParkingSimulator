Purpose:
These programs are meant to demonstrate a parking lot simulation
in which cars enter a parking lot and pay their fees according to
how long they have stayed and the rate of the lot.

List of files:
- parkingSimulator.c
- simulator2.c
- Readme.txt

Instructions to run program:
For parkingSimulator.c
gcc -o file parkingSimulator.c
./file

For simulator2.c
gcc -o file2 simulator2.c
./file2
